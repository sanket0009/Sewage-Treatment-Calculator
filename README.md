# Sewage Treatment Design Calculator

## 📌 Overview
This project is a web-based application that helps in designing and calculating parameters for a complete sewage treatment plant. It automates engineering calculations and provides accurate results instantly.

## 🚀 Features
- Bar Screen Design
- Grit Chamber Calculation
- Primary Sedimentation Tank Design
- Activated Sludge Process
- Secondary Clarifier Design
- Oxidation Pond Calculation

## 🛠️ Tech Stack
- Frontend: HTML, CSS
- Backend: Flask (Python)
- Database: SQLite

## 📊 Modules
1. Bar Screen
2. Grit Chamber
3. Primary Sedimentation Tank
4. Activated Sludge Process
5. Secondary Clarifier
6. Oxidation Pond
